Petunjuk Penggunaan:

  1. Salin data scatter dari dokumen (termasuk User ID, link screenshot, kode tiket, nama pemain, dan bettingan)
  2. Tempel data yang telah disalin ke dalam textarea "Data Scatter"
  3. Klik tombol "Proses Data" untuk mengekstrak informasi
  4. Data yang diekstrak akan ditampilkan di sisi kanan
  5. Klik gambar screenshot untuk melihatnya dalam ukuran besar
  6. Gunakan tombol "Salin" untuk menyalin data tertentu
  7. Gunakan tombol "Salin Semua Link Sekaligus" untuk menyalin semua link screenshot dalam satu baris
  8. Gunakan tombol "Salin Semua Data" untuk menyalin semua data termasuk link
